# 📝 Blog-Project

## 📌 프로젝트 소개
나만의 블로그 서비스를 개발하면서 백엔드의 전반적인 기술 흐름을 경험하는 것이 목표입니다.  
기획, DB 설계, 인증, CRUD API, 배포까지 실제 서비스를 만드는 과정을 학습합니다.

## 🎯 목표
- 웹 서비스 아키텍처 경험
- REST API 설계 및 구현
- 인증/인가(JWT) 흐름 학습
- 프로젝트 문서화 및 일정 관리 실습
- 경험 및 학슴 내용 정리

## ⚙️ 기술 스택
- Java 17
- Spring Boot
- Spring Security
- JPA (Hibernate)
- MySQL
- JWT
- GitHub Projects (진행 로그 관리)

## 🧩 주요 기능 (MVP)
- 회원가입 / 로그인 (JWT 기반)
- 게시글 목록 조회
- 게시글 상세 조회
- 게시글 작성 / 수정 / 삭제
- 로그인 유저와 작성자 매칭
